A Dungeons and Dragon character manager. Allows players to quickly refer to important stats, view store items, and cast spells.

Programming Experience

I've never programmed in either of the above programming languages. Prior to starting this project, I reviewed some examples of React and used a beginner's tutorial to better understand the project. My partner on this project, Ryan Vreeke, has prior experience in SQL. I left the majority of the SQL in his capable hands, while taking this as an opportunity to learn more about SQL.

Planning and UML

I started with what I wanted the user experience to be and features I wanted to include. Below is a list of the features I intend on including:

  • Select Current Character
  • View all important stats
  • Remove/Add Health
  • Cast Spells
  • Update Inventory
